## 综述

calendar-deprecated 是 KISSY <= 1.3 的内置日历组件。

## 快速使用

### 初始化组件

    S.use('gallery/calendar-deprecated/1.0/index', function (S, Calendar) {
         var calendar-deprecated = new Calendar();
    })
